
# QMediaVSIX
Quickly play, pause and skip your currently playing media with this Visual Studio 2019 extension. Includes (partial) Japanese translations.


このVisual Studio 2019の拡張機能を使えば、現在再生中のメディアを素早く再生、一時停止、スキップできます。
